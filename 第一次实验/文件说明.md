## 🚀 快速开始流程

### 第一步：安装依赖
```bash

```

编辑 `api_config.py`，填入你的 API Key

## 🔍 常见问题排查

### 问题 1：无法运行程序

**检查步骤**：
1. Python 版本是否 >= 3.8
2. 是否安装了 requests 包
3. 是否在正确的目录下运行

### 问题 2：API 调用失败

**检查步骤**：
1. API Key 是否正确
2. 网络连接是否正常
3. API 额度是否充足
4. 运行 `python test_api.py` 测试

### 问题 3：结果未保存

**检查步骤**：
1. 查看当前目录是否有 JSON 文件
2. 检查文件权限
3. 查看程序是否有错误输出
