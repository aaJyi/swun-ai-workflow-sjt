### 三、实验原理

#### 3.1 AI 大模型概述

AI 大模型（Large Language Model, LLM）是基于深度学习技术，通过在大规模语料库上进行预训练，获得强大的语言理解和生成能力的人工智能模型。这类模型能够完成对话、文本创作、代码生成、逻辑推理等多种任务。

#### 3.2 API 调用原理

API（Application Programming Interface，应用程序编程接口）是不同软件系统之间进行通信的桥梁。AI 大模型 API 调用的一般流程如下：

1. **身份认证**：通过 API Key 进行身份验证
2. **请求构建**：将用户输入按照 API 要求的格式组织成请求体
3. **发送请求**：通过 HTTP/HTTPS 协议将请求发送到 API 服务器
4. **接收响应**：解析服务器返回的 JSON 格式响应
5. **结果展示**：将 AI 生成的内容呈现给用户

#### 3.3 RESTful API

本实验使用的 AI 服务都采用 RESTful API 风格，具有以下特点：
- 使用标准的 HTTP 方法（GET、POST 等）
- 数据格式通常为 JSON
- 无状态通信，每次请求包含所有必要信息
- 统一的资源标识（URL）

### 四、实验内容与步骤

#### 4.1 实验准备

**步骤 1：获取 API 密钥**

1. 访问智谱 AI 开放平台：https://open.bigmodel.cn/
2. 注册账号并完成实名认证
3. 进入"个人中心" -> "API 密钥管理"
4. 创建新的 API Key，复制并保存

**步骤 2：配置开发环境**

```bash
# 安装必要的 Python 库
pip install requests
```

**步骤 3：配置 API 密钥**

编辑 `api_config.py` 文件，将获取的 API Key 填入对应位置：

```python
ZHIPU_AI_CONFIG = {
    'api_key': '你的 API Key',  # 替换为实际密钥
    'model': 'glm-4',
    'base_url': 'https://open.bigmodel.cn/api/paas/v4',
}
```

#### 4.2 实验任务

**任务一：智能对话**

- **任务描述**：与 AI 进行对话，了解人工智能发展历程
- **输入**："请用简洁的语言介绍人工智能的发展历程，不超过 200 字。"
- **预期目标**：AI 能够准确、简洁地介绍 AI 发展的关键节点

**任务二：文本创作**

- **任务描述**：让 AI 以"人工智能与未来生活"为题创作文本
- **输入**："请以'人工智能与未来生活'为题，写一段 150 字左右的短文。"
- **预期目标**：AI 能够生成连贯、有逻辑的短文

**任务三：逻辑推理**

- **任务描述**：测试 AI 的逻辑推理能力
- **输入**："如果所有的猫都喜欢吃鱼，咪咪是一只猫，那么咪咪喜欢吃鱼吗？请解释你的推理过程。"
- **预期目标**：AI 能够正确进行三段论推理并解释过程

**任务四：代码生成**

- **任务描述**：让 AI 生成 Python 代码
- **输入**："请用 Python 写一个函数，计算列表中所有偶数的和。"
- **预期目标**：AI 能够生成正确、可运行的代码

**任务五：知识问答**

- **任务描述**：测试 AI 对专业概念的理解和解释能力
- **输入**："什么是机器学习？它和深度学习有什么区别？"
- **预期目标**：AI 能够准确解释概念并说明区别
